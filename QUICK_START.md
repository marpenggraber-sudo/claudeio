## ⚠️ 注意事项

### **1. Redis 可选**
- Redis 不运行也没问题
- 如果想使用 Redis，启动后重启后端即可

### **2. 验证码登录问题**
- 验证码登录依然会报 10004 错误（网易云风控）
- **解决方案**: 使用二维码登录（已修复）

### **3. api-enhanced 必须运行**
确保 Docker 容器正在运行：
```bash
docker ps | findstr ncm-api
```

如果未运行：
```bash
docker start ncm-api
```

## 🆘 遇到问题？

### **后端启动失败**
检查 MySQL 是否运行：
```bash
# 检查 MySQL 服务
net start | findstr MySQL
```

### **二维码生成失败**
检查 api-enhanced 是否运行：
```bash
curl http://localhost:3000/login/qr/key
```

### **扫码后没反应**
查看后端日志，确认是否有错误：
```bash
# 查看最后 50 行日志
tail -50 claudeio_backend/logs/application.log
```
